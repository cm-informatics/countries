This app displays a list of countries of the world.
The new UISearchController is implemented to find a specific country very fast.
If the country has states its possible to display them by tapping on this county.

Techniques:

- JSON-Parsing from a local file
- Programmatically implemented UISearchController
- Programmatically created UITableViewController and segue for the states
